# Hedge Fund Server

A simple NestJS server for hedge fund operations.

## Installation

```bash
pnpm install
```

## Running the app

```bash
# development
pnpm run start:dev

# build
pnpm run build

# production mode
pnpm run start:prod
```

## API

- `GET /` - Returns "Hello World!"

The server runs on `http://localhost:3000`
